## 🚀 **Quick Start Commands**

### **Installation**
```bash
git clone https://github.com/yourusername/ultimate-tech-detection.git
cd ultimate-tech-detection
pip install -r requirements.txt
cp env.example .env
# Edit .env with your API keys
```

### **Running**
